# MinimalPackage

**MinimalPackage** is a Swift package designed for modular development and automated distribution. This repository serves as a template or example for managing Swift packages with integrated DocC documentation and XCFramework releases.

## Repository Structure

```text
.
├── minimal-package/    # Main Swift Package source code
├── scripts/            # Automation scripts (Build, Docs, Release)
├── docs/               # Generated static documentation site
├── tutorials/          # DocC tutorials and guides
└── README.md           # This file
```

## Quick Start

### 🛠 Development

The core logic resides in the `minimal-package/` directory. You can open `Package.swift` in Xcode or use the Swift CLI.

### 📚 Documentation

We use Swift DocC for documentation. You can generate or preview it using our scripts:

```bash
# Preview documentation locally
./scripts/generate-docs.sh --preview
```

The documentation is also hosted on GitHub Pages:
`https://<user>.github.io/<repo>/documentation/minimalpackage`

### 🚀 Automation & Releases

The project includes several scripts to simplify your workflow:

- **`./scripts/archive.sh`**: Builds the project into an XCFramework for distribution.
- **`./scripts/generate-docs.sh`**: Builds and serves the DocC documentation site.
- **`./scripts/release.sh <version>`**: Orchestrates a complete release, including documentation, binary assets, and GitHub publishing.

For more details on these scripts, see the [Scripts Documentation](./scripts/README.md).
